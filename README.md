# Gold Price Prediction using Random Forest

## 📌 Overview
This project uses a **Random Forest Regressor** to predict gold prices based on historical data. It involves **data preprocessing, visualization, correlation analysis, and model training** to improve accuracy.

## 📂 Dataset
- The dataset used: **`gld_price_data.csv`**
- It includes historical gold price data and related economic indicators.

## 📊 Features Used
- Date (converted to datetime format)
- Various economic indicators affecting gold prices
- Target variable: **GLD Price (Gold Price in USD)**

## 🛠️ Technologies Used
- **Python**
- **Pandas, NumPy** (Data processing)
- **Matplotlib, Seaborn** (Data visualization)
- **Scikit-learn** (Machine Learning - Random Forest Regressor)

## 📈 Results
- The model evaluates performance using **R² (R-squared) score**.
- Data visualization includes **correlation heatmaps, distribution plots, and actual vs predicted price graphs**.

## 🔥 Future Improvements
- Try using **deep learning models** (LSTMs or neural networks).
- Incorporate **more financial indicators** for better predictions.
